# Tetris_Game_React_TS
A browser-based Tetris game built with React, TypeScript and Vite.

## Features
- 10x20 game board
- Tetromino movement and rotation
- Collision detection
- Line clearing
- Score tracking
- Game over state

## Controls
- Left Arrow — move left
- Right Arrow — move right
- Down Arrow — move down
- Up Arrow — rotate piece

## Tech Stack
- React
- TypeScript
- Vite
- CSS
